# sql codes

Login to MySQL shell:
```
mysql -u root -p
```

Create a database:
```
CREATE DATABASE my_db;
```

Delete a database:
```
DROP DATABASE db_name;
```

Show databases list:
```
SHOW DATABASES;
```

Your command line should look something like this:
```
 mysql> SHOW DATABASES;
+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |
| performance_schema |
| test               |
+--------------------+
4 rows in set (0.01 sec)
```
